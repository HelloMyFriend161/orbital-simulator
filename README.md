# Orbit Simulator

Little side project of mine coded in lua using the LÖVE2d framework for the rendering.

It is advised that you know at least a little about how LÖVE works to be able to run this.

# Plans
- Cartesian/Keplerian conversion
- Multiple celestial bodies
- Satellites or something
- Complex Simulations

The rendering won't change too much as it is just a simulation, not a game, but whatever.
